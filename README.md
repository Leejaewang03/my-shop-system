# AI Shopping Mall System

## 프로젝트 소개
React 기반의 쇼핑몰 웹 시스템입니다.

생성형 AI(ChatGPT)의 도움을 받아 구현하였으며,
상품 검색, 장바구니 기능 등을 제공합니다.

---

## 사용 기술

- React
- Vite
- GitHub Actions
- AWS Amplify

---

## 주요 기능

- 상품 목록 조회
- 상품 검색
- 장바구니 추가
- 총 금액 계산
- 반응형 UI

---

## Github Actions CI/CD

Github Actions를 활용하여
코드 push 시 자동 빌드가 수행되도록 구성하였습니다.

### Workflow 기능

- npm install
- React build 자동 수행
